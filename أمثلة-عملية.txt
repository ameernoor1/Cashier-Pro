📖 المثال 1: مستخدم جديد - البداية من الصفر
═══════════════════════════════════════════════════════════════════

السيناريو:
----------
محمد يريد استخدام نظام كاش برو لإدارة محله التجاري

الخطوات:
--------

الخطوة 1: زيارة الموقع
   1. محمد يفتح: landing.html
   2. يشاهد صفحة جذابة تعرض مميزات النظام
   3. يقرأ عن الخطط الثلاثة:
      • مجاني (30 يوم)
      • شهري (25,000 د.ع)
      • سنوي (225,000 د.ع)

الخطوة 2: اختيار الخطة
   1. محمد يقرر البدء بالخطة المجانية
   2. يضغط على زر "ابدأ التجربة المجانية"
   3. يتم نقله إلى: register.html

الخطوة 3: التسجيل
   1. محمد يملأ النموذج:
      • الاسم: محمد أحمد
      • الهاتف: 07701234567
      • الدولة: العراق
      • المدينة: بغداد
      • الشارع: الكرادة
   2. يضغط "إنشاء الحساب والبدء"
   3. يظهر له: "جاري إنشاء حسابك..."

الخطوة 4: الدخول للنظام
   1. يتم إنشاء معرف فريد: USER-1234567890-ABC123
   2. يتم حفظ البيانات في Firebase
   3. يتم نقله تلقائياً إلى: app.html
   4. محمد الآن يستطيع استخدام جميع المميزات!

النتيجة:
--------
✅ محمد لديه حساب نشط لمدة 30 يوم
✅ جميع المميزات متاحة
✅ بياناته محفوظة بأمان
✅ لديه 30 يوم للتجربة

📖 المثال 2: مستخدم حالي - تجديد الاشتراك
═══════════════════════════════════════════════════════════════════

السيناريو:
----------
فاطمة استخدمت النظام لمدة 30 يوم، والآن انتهت فترتها التجريبية

الخطوات:
--------

اليوم 28 (تنبيه):
   1. فاطمة تفتح النظام
   2. يظهر لها تنبيه أصفر في الأعلى:
      "⚠️ تنبيه: اشتراكك على وشك الانتهاء!
       المتبقي: 2 أيام فقط"
   3. فاطمة تقرر الاشتراك الشهري

اليوم 31 (انتهاء الاشتراك):
   1. فاطمة تحاول فتح النظام
   2. تظهر نافذة كبيرة (لا يمكن إغلاقها):
      ┌─────────────────────────────────────┐
      │       ⏰ انتهى اشتراكك!           │
      │                                     │
      │  للاستمرار، يرجى تجديد الاشتراك  │
      │                                     │
      │  [حقل إدخال رمز التفعيل]          │
      │  [تفعيل الاشتراك]                 │
      │  [شراء اشتراك جديد]               │
      └─────────────────────────────────────┘
   
   3. فاطمة تتصل بالمسؤول وتطلب رمز تفعيل
   4. المسؤول يعطيها: MONTHLY-ABC123XYZ456
   5. فاطمة تدخل الرمز وتضغط "تفعيل"
   6. ✅ يظهر لها: "🎉 تم تفعيل اشتراكك بنجاح!"
   7. الصفحة تتحدث تلقائياً
   8. فاطمة الآن لديها 30 يوم إضافية!

النتيجة:
--------
✅ تم تجديد اشتراك فاطمة
✅ لديها 30 يوم جديدة
✅ جميع المميزات عادت للعمل
✅ الرمز أصبح "مستخدم" ولا يمكن استخدامه مرة أخرى

📖 المثال 3: المدير - إنشاء رموز التفعيل
═══════════════════════════════════════════════════════════════════

السيناريو:
----------
أحمد هو مدير النظام، يريد إنشاء 5 رموز شهرية و 2 رمز سنوي

الخطوات:
--------

الخطوة 1: الدخول إلى لوحة المدير
   1. أحمد يفتح: admin.html
   2. يرى لوحة تحكم شاملة مع:
      • إحصائيات عامة
      • قائمة بجميع المشتركين
      • أدوات إنشاء الرموز

الخطوة 2: إنشاء رموز شهرية
   1. يذهب إلى تبويب "إنشاء رموز التفعيل"
   2. يختار: "نوع الاشتراك: شهري"
   3. يضغط "إنشاء رمز" 5 مرات
   4. يحصل على:
      • MONTHLY-ABC123XYZ456
      • MONTHLY-DEF789UVW012
      • MONTHLY-GHI345RST678
      • MONTHLY-JKL901OPQ234
      • MONTHLY-MNO567KLM890

الخطوة 3: إنشاء رموز سنوية
   1. يختار: "نوع الاشتراك: سنوي"
   2. يضغط "إنشاء رمز" مرتين
   3. يحصل على:
      • YEARLY-ABC123XYZ456
      • YEARLY-DEF789UVW012

الخطوة 4: حفظ الرموز
   1. أحمد ينسخ جميع الرموز
   2. يحفظها في ملف Excel أو Word
   3. يضع ملاحظات: التاريخ، الغرض، المستفيد

الخطوة 5: توزيع الرموز
   1. أحمد يعطي رمز شهري لمحمد
   2. يعطي رمز سنوي لفاطمة
   3. يحتفظ بباقي الرموز للمستقبل

النتيجة:
--------
✅ تم إنشاء 7 رموز جديدة
✅ جميع الرموز محفوظة في Firebase
✅ يمكن تتبع استخدام كل رمز
✅ الرموز جاهزة للتوزيع

📖 المثال 4: المدير - متابعة المشتركين

----------
أحمد يريد معرفة حالة جميع المشتركين

الخطوات:
--------

الخطوة 1: فتح تبويب المشتركين

   2. يضغط على تبويب "عرض قاعدة البيانات"

الخطوة 2: مراجعة القائمة
   أحمد يرى قائمة بجميع المشتركين:
   
   ┌────────────────────────────────────────────┐
   │ محمد أحمد                                 │

   │ 🆔 USER-1234567890-ABC123                │

   │ 🏷️ تجريبي مجاني                        │

   ┌────────────────────────────────────────────┐
   │ فاطمة علي                                 │
   │ 📱 07709876543                            │
   │ 📍 العراق - بصرة - الجمهورية           │
   │ 🆔 USER-0987654321-XYZ789                │
   │ ✅ نشط (28 يوم متبقي)                  │
   │ 🏷️ شهري                                 │
   │ 🔑 MONTHLY-ABC123XYZ456                  │
   └────────────────────────────────────────────┘
   
   ┌────────────────────────────────────────────┐
   │ علي حسن                                   │
   │ 📱 07705551234                            │

   │ 🏷️ تجريبي مجاني                        │
   └────────────────────────────────────────────┘

الخطوة 3: التحليل
   أحمد يلاحظ:
   • محمد: اشتراكه ينتهي بعد 25 يوم
   • فاطمة: اشتراكها نشط لمدة 28 يوم
   • علي: اشتراكه انتهى ويحتاج تجديد

الخطوة 4: الإجراء
   1. أحمد يتصل بعلي
   2. يعرض عليه اشتراك شهري أو سنوي
   3. علي يوافق على الاشتراك السنوي
   4. أحمد يعطيه رمز: YEARLY-ABC123XYZ456

النتيجة:
--------
✅ أحمد لديه رؤية كاملة لجميع المشتركين
✅ يمكنه المتابعة والتواصل المباشر
✅ استباقية في التجديد قبل الانتهاء

📖 المثال 5: سيناريو كامل - من البداية للنهاية
═══════════════════════════════════════════════════════════════════

السيناريو:
----------
سارة تريد فتح محل للملابس وتحتاج نظام كاشير

اليوم 1 - الاكتشاف:
──────────────────────────────────────────────────────────────────
   1. سارة تبحث في Google عن "نظام كاشير عراقي"
   2. تجد رابط "كاش برو"
   3. تفتح landing.html
   4. تقرأ عن المميزات:
      ✓ سرعة فائقة
      ✓ تقارير شاملة
      ✓ إدارة مخزون
      ✓ طباعة فواتير
      ✓ أمان عالي
   5. تعجبها الخطة المجانية (30 يوم)
   6. تضغط "ابدأ التجربة المجانية"

اليوم 1 - التسجيل:
──────────────────────────────────────────────────────────────────
   1. تملأ بياناتها:
      • الاسم: سارة خالد
      • الهاتف: 07703334444
      • الدولة: العراق
      • المدينة: الموصل
      • الشارع: الرشيدية
   2. تضغط "إنشاء الحساب"
   3. تدخل النظام

اليوم 1-7 - الإعداد:
──────────────────────────────────────────────────────────────────
   1. سارة تضيف منتجاتها:
      • فستان صيفي (45,000 د.ع)
      • بلوزة (25,000 د.ع)
      • بنطال جينز (35,000 د.ع)
      • ... 200 منتج آخر
   2. تنظم المنتجات في فئات
   3. تجرب عمليات البيع
   4. تطبع بعض الفواتير التجريبية

اليوم 8-30 - الاستخدام الفعلي:
──────────────────────────────────────────────────────────────────
   1. سارة تستخدم النظام يومياً
   2. تبيع في المتوسط 20-30 قطعة يومياً
   3. تتابع المخزون والأرباح
   4. تطبع فواتير للزبائن
   5. النظام يعمل بسلاسة 100%

اليوم 28 - التنبيه:
──────────────────────────────────────────────────────────────────
   1. سارة تفتح النظام
   2. يظهر تنبيه: "⚠️ اشتراكك ينتهي بعد يومين!"
   3. سارة مقتنعة جداً بالنظام
   4. تقرر الاشتراك السنوي (وفر 3 أشهر!)

اليوم 29 - التواصل:
──────────────────────────────────────────────────────────────────
   1. سارة تتصل بالدعم الفني
   2. تطلب اشتراك سنوي
   3. تدفع: 225,000 دينار عراقي
   4. تستلم رمز: YEARLY-ABC123XYZ456

اليوم 30 - التفعيل:
──────────────────────────────────────────────────────────────────
   1. سارة تفتح النظام
   2. تدخل الرمز في نافذة التجديد
   3. يتم التفعيل فوراً
   4. لديها الآن سنة كاملة!

اليوم 31-395 - النجاح:
──────────────────────────────────────────────────────────────────
   1. سارة تستخدم النظام يومياً
   2. محلها يزدهر
   3. المبيعات تزيد 40%
   4. إدارة أفضل، أرباح أكثر!

النتيجة النهائية:
--------
✅ سارة تحولت من مستخدمة جديدة إلى عميلة مخلصة
✅ النظام ساعدها في تحسين إدارة محلها
✅ وفرت الوقت والجهد
✅ زادت الأرباح بشكل ملحوظ

💡 نصائح من الأمثلة الواقعية

للمستخدمين الجدد:
──────────────────────────────────────────────────────────────────
✓ استفد من الفترة التجريبية المجانية
✓ أضف منتجاتك بالتدريج
✓ جرب جميع المميزات
✓ لا تنتظر حتى آخر يوم للتجديد

للمستخدمين الحاليين:
──────────────────────────────────────────────────────────────────
✓ انتبه للتنبيهات
✓ جدد قبل الانتهاء بأسبوع
✓ الاشتراك السنوي أوفر
✓ احتفظ برمز التفعيل في مكان آمن

للمدراء:
──────────────────────────────────────────────────────────────────
✓ أنشئ رموز إضافية احتياطياً
✓ تابع المشتركين باستمرار
✓ تواصل مع المشتركين قبل انتهاء اشتراكهم
✓ احتفظ بسجل لجميع الرموز المستخدمة

═══════════════════════════════════════════════════════════════════
❓ أسئلة شائعة مع أمثلة
═══════════════════════════════════════════════════════════════════

س: ماذا يحدث عند انتهاء اشتراكي؟
ج: مثال: محمد حاول فتح النظام بعد انتهاء اشتراكه، ظهرت له نافذة
   تطلب منه تجديد الاشتراك. لا يمكنه إضافة منتجات أو بيع حتى
   يجدد اشتراكه.

س: هل يمكنني استخدام رمز تفعيل مرتين؟
ج: مثال: فاطمة حاولت إعطاء رمزها لصديقتها، لكن الرمز كان
   "مستخدم" بالفعل. كل رمز يعمل مرة واحدة فقط.

س: كيف أعرف كم يوم متبقي في اشتراكي؟
ج: مثال: سارة تفتح النظام وتنظر في الزاوية العلوية، تجد
   "الاشتراك: نشط - 15 يوم متبقي".

س: هل بياناتي آمنة؟
ج: مثال: علي سأل نفس السؤال. جميع البيانات محفوظة في
   Firebase (خوادم Google الآمنة) + نسخة محلية في الجهاز.

س: ماذا أفعل إذا فقدت رمز التفعيل؟
ج: مثال: أحمد فقد رمزه، اتصل بالمدير الذي فحص قاعدة
   البيانات وأعطاه رمز جديد.
